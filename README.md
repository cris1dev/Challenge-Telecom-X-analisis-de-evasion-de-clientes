# Challenge-Telecom-X-analisis-de-evasion-de-clientes
Segundo Desafío Alura Latam ciencia de datos, evolución del proyecto
Antecedentes: la empresa Telecom X enfrenta una alta tasa de cancelación de servicios y necesita comprender los factores que llevan a la pérdida de clientes.
Objetivos específicos: como parte del proyecto "Churn de Clientes", se dará solución al requiermiento a través de las acciones de; recopilar, procesar y analizar los datos, utilizando Python en plataforma Google Colab y sus principales bibliotecas para extraer información valiosa y generar los respectivos gráficos representativos.
Objetivo general: a partir del análisis generado, el equipo de Data Science podrá avanzar en modelos predictivos y desarrollar estrategias para reducir la evasión.
